# Portfolio Requirements

## Task Title: Craft Your Developer Profile

### Objective
Create a professional and responsive portfolio website that showcases skills, experience, and projects effectively.

---

## Design Reference
- **Style:** Dark theme similar to [Dribbble - Frontend Developer Portfolio](https://dribbble.com/shots/15291933-Frontend-Developer-portfolio-website-design)
- **Color Scheme:** Dark navy/purple with gradient accents
- **User:** Female developer (use female placeholder images)

---

## Task Requirements

### 1. Navigation Bar (Navbar)
- [ ] Fully responsive navigation bar
- [ ] Easy access to all sections of the portfolio from the navbar

### 2. Hero Section - Designation and Introduction
- [ ] Display professional designation (e.g., Frontend Developer, Web Developer, Full Stack Developer)
- [ ] Include a professional photo (placeholder for now, user will provide)
- [ ] Resume download button - clearly visible, allows visitors to view/download resume with single click
  - *Note: Keep button clickable even if resume not ready yet*

### 3. Social Links
- [ ] Embed buttons linking to relevant social media profiles:
  - GitHub
  - LinkedIn
  - Twitter
  - Facebook

### 4. About Me Section
- [ ] Detailed introduction including:
  - Programming journey
  - Type of work enjoyed
  - Hobbies/interests outside programming (e.g., sports, painting)
- [ ] Showcase personality in this section

### 5. Skills Section
- [ ] Display skills in visually appealing graphical format
- [ ] Categorize skills (e.g., Frontend, Backend, Tools)

### 6. Educational Qualification
- [ ] Mention educational background in detail (if completed education above HSC)
- [ ] Can be skipped if not applicable

### 7. Experience Section
- [ ] Add relevant professional experience (if available)
- [ ] Can be skipped if not applicable

### 8. Projects Section (Minimum 3 Projects)
- [ ] Showcase at least 3 best projects in card format
- [ ] Each project card must contain:
  - Project name
  - Project image
  - "View More / Details" button

- [ ] Clicking "View More / Details" leads to detailed project page containing:
  - Main technology stack used
  - Brief description
  - Live project link
  - GitHub repository link (client only)
  - Challenges faced while developing
  - Potential improvements and future plans

### 9. Contact Information
- [ ] Easy way for visitors to contact directly
- [ ] Must include:
  - Email address
  - Phone number
  - WhatsApp number (optional)

### 10. Footer
- [ ] Simple and elegant footer for better presentation

### 11. Responsive Design
- [ ] Full responsiveness across all screen sizes:
  - Desktop
  - Laptop
  - Tablet
  - Mobile
- [ ] Polished and professional color scheme

---

## Technology Stack
- **Framework:** React + Vite
- **Styling:** CSS with dark theme
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Routing:** React Router (for project detail pages)
